# Pipeline Reports

This directory is populated by Jenkins and archived as build artifacts.

- `reports/sast/` SonarQube-related artifacts/logs
- `reports/sca/` OWASP Dependency-Check JSON/HTML reports
- `reports/image/` Trivy image scan report
- `reports/secret/` Gitleaks report
- `reports/sbom/` Syft CycloneDX SBOM
- `reports/deploy/` ECS deployment evidence and rendered task definition
